Get the free variables of the expression.

Returns:the free variables
Return type:set

>>> from lapp import *
>>> from labs import *

>>> LambdaExp(x).freeVar() == {"x"}

True
>>> y = LambdaVar("y")
>>> xy = LambdaApp(x, y)
>>> LambdaExp(xy).freeVar() == {"x", "y"}
True
>>> LambdaExp(LambdaAbs("x", xy)).freeVar() == {"y"}
True
>>> LambdaExp(LambdaAbs("y", xy)).freeVar() == {"x"}
True
